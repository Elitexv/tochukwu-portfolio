# Mechanical Engineer Resume Website

A modern, high-class resume website for a mechanical engineer with beautiful UI/UX design.

## Features

- **Modern Design**: Clean, professional interface with smooth animations
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Interactive Elements**: Smooth scrolling, animated skill bars, hover effects
- **Complete Sections**:
  - Hero section with professional photo
  - About me with statistics
  - Professional experience timeline
  - Education and certifications
  - Technical skills with progress bars
  - Featured projects portfolio
  - Contact form

## Technologies Used

- HTML5
- CSS3 (with modern features like gradients, animations, and flexbox/grid)
- JavaScript (ES6+)
- Font Awesome icons
- Google Fonts (Inter)

## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or dependencies required - it's a pure HTML/CSS/JS website

## Customization

### Personal Information
Edit the following in `index.html`:
- Name, title, and description in the hero section
- About section content
- Experience entries
- Education details
- Skills and proficiency levels
- Project descriptions
- Contact information

### Images
Replace the Unsplash image URLs with your own images:
- Hero background image
- Profile photo
- About section image
- Project images

### Colors
Modify the CSS variables in `styles.css` (in the `:root` selector) to change the color scheme:
```css
--primary-color: #2563eb;
--secondary-color: #0ea5e9;
--accent-color: #06b6d4;
```

### Skills
Adjust skill proficiency percentages in the HTML by modifying the `width` style attribute on `.skill-progress` elements.
